The Model

Ensemble Approach

We blend two complementary methods:

Expert System (60%): Hand-tuned weights analyzing pitcher quality (24%), team offense (16%), bullpen strength (10%), home field advantage, and Pythagorean win percentage. Conservative and well-calibrated.

Machine Learning (40%): Logistic regression + gradient boosting trained on 2,426 games from the 2025 season. Learned optimal feature weights from real outcomes.

What We Analyze

Starting Pitchers

Season stats (ERA, WHIP, K/9, BB/9, HR/9) + recent form (last 5 starts) + career vs opponent + park history.

Team Strength

Offensive firepower (OPS, runs/game), bullpen quality (ERA, saves), true talent (Pythagorean win %), home/road splits.

Context & Validation

Park factors (Coors = 1.30, Oracle = 0.92), home field advantage, Vegas lines (to validate our edge).

Daily Process

10 AM ET Every Day

1. Fetch today's games and probable pitchers from MLB Stats API
2. Pull current season stats for all teams and pitchers
3. Fetch Vegas lines from DraftKings/ESPN
4. Run both models and blend predictions (60% expert, 40% ML)
5. Filter for 64%+ confidence only
6. Publish picks 3+ hours before first pitch
7. Track results and update metrics

Quality Standards

No Look-Ahead Bias: Backtest uses zero future data. Pitcher stats computed only from games before each prediction date.

Realistic Odds: ROI calculated using actual moneyline odds with vig, not flat -110.

Honest Thresholds: 64% threshold gives ~2-4 picks/day with 75%+ accuracy. Some days have zero picks.

Full Transparency: Every pick is timestamped. Every result is tracked. Bad days are published.
