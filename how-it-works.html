How It Works

Data-driven predictions. No gut feelings. No hype. Just probabilities backed by machine learning.

🧠 The Model

We use an ensemble of two approaches:

📊

Hand-Tuned Expert System (60%)

Built by analyzing thousands of games. Weights factors like pitcher quality (24%), team offense (16%), bullpen strength (10%), home field advantage, and run differential.

🤖

Machine Learning (40%)

Logistic regression + gradient boosting trained on 2,426 games from the 2025 season. Learned optimal weights from real outcomes, not guesses.

✅ Why Blend Both?

The ML model is data-driven but can be overconfident early in the season. The expert system is conservative and well-calibrated. Blending them (60/40) gives us the best of both: smart feature selection from ML, stability from human expertise.

🎯 What We Analyze

⚾

Starting Pitchers

Season stats (ERA, WHIP, K/9, BB/9, HR/9) + recent form (last 5 starts) + career vs opponent + park history.

🏟️

Team Strength

Offensive firepower (OPS, runs/game), bullpen quality (ERA, saves), true talent (Pythagorean win %), home/road splits.

📍

Context

Park factors (Coors = 1.30, Oracle = 0.92), home field advantage, rest days, Vegas lines (to validate edge).

📈 Daily Process

Every Morning at 10 AM ET:

  1. Fetch today's games and probable pitchers from MLB Stats API
  2. Pull current season stats for all teams and pitchers
  3. Fetch Vegas lines from DraftKings/ESPN
  4. Run both models (v2 + v3) and blend predictions (60/40)
  5. Filter for HIGH confidence only (64%+ win probability)
  6. Publish picks 3+ hours before first pitch (transparency)
  7. Track results and update accuracy metrics

🔬 Quality Standards

No Look-Ahead Bias

Backtest uses zero future data. Pitcher stats computed only from games before each prediction date. Team stats from prior season. No cheating.

Realistic Odds

ROI calculated using actual moneyline odds with vig, not flat -110. If we pick a favorite at -150, we account for the real payout (not fantasy numbers).

Honest Thresholds

We tested 62%, 65%, 68% thresholds. Settled on 64% because it gives ~2-4 picks/day with 75%+ accuracy. Some days there are ZERO picks if nothing clears the bar.

Full Transparency

Every pick is timestamped. Every result is tracked. Bad days are published (see Track Record). No cherry-picking.

❓ What This ISN'T

  • NOT a guaranteed winner — variance exists, upsets happen
  • NOT based on "insider info" or gut feelings — pure data
  • NOT trying to pick every game — selective is better
  • NOT claiming 90%+ win rates — that's a red flag
  • NOT hiding bad picks — full history is public

📊 Proven Results

Backtested on the full 2025 season (2,426 games). HIGH confidence picks went 66.2% with +9.1% ROI. Live tracking since March 2026 shows 75% accuracy and +43% ROI (small sample, but consistent with backtest).

When we show "+4.2% vs market" on a pick, that means we think Vegas is wrong and there's real edge. That's the signal worth betting on.
