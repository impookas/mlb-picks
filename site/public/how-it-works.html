How It Works

Data-driven predictions. No gut feelings. Just probabilities backed by machine learning.

The Model

We blend two approaches:

Expert System (60%): Hand-tuned weights analyzing pitcher quality (24%), team offense (16%), bullpen strength (10%), home field advantage, and run differential.

Machine Learning (40%): Logistic regression + gradient boosting trained on 2,426 games from 2025. Learned optimal weights from real outcomes.

What We Analyze

Starting Pitchers: ERA, WHIP, K/9, BB/9, HR/9, recent form (last 5 starts), career vs opponent.

Team Strength: Offensive power (OPS, runs/game), bullpen quality, Pythagorean win %, home/road splits.

Context: Park factors, Vegas lines (to validate edge), rest days.

Daily Process

10 AM ET Every Day:

1. Fetch games and pitchers from MLB Stats API
2. Pull current season stats
3. Fetch Vegas lines from DraftKings/ESPN
4. Run both models and blend (60/40)
5. Filter for 64%+ confidence only
6. Publish picks 3+ hours before first pitch
7. Track results and update metrics

Quality Standards

No Look-Ahead Bias: Backtest uses zero future data. Pitcher stats computed only from prior games.

Realistic Odds: ROI uses actual moneyline odds with vig, not fantasy -110.

Honest Thresholds: 64% threshold gives ~2-4 picks/day with 75%+ accuracy. Some days have ZERO picks.

Full Transparency: Every pick is timestamped. Every result is tracked. Bad days are published.

Proven Results

Backtested on 2,426 games from 2025 season. HIGH picks: 66.2% win rate, +9.1% ROI.

Live tracking since March 2026: 75% accuracy, +43% ROI (early sample, but consistent with backtest).

When we show "+4.2% vs market", we think Vegas is wrong and there's real edge.
